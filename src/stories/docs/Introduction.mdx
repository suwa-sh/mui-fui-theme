# FUI Theme

**FUI (Futuristic User Interface) Theme** は、MUI (Material-UI) をベースとした未来的なデザインシステムです。J.A.R.V.I.S. スタイルのHUD/FUIデザインを実現します。

## 特徴

- **シャープなエッジ**: 角丸を排除したシャープなデザイン
- **グローエフェクト**: ダークモードで映える光るエフェクト
- **モノスペースフォント**: JetBrains Mono を採用したテクニカルな雰囲気
- **ステージカラー**: 13段階のカラースキーム（stage1-4: グラデーション、stage4→13→1: 色相環）
- **ダーク/ライトモード対応**: 両モードで最適化されたカラーパレット
- **レスポンシブ対応**: ブレークポイント別のサイズ定義
- **CodeMirror統合**: エディタ用テーマ
- **CSSアニメーション**: fadeIn, pulse, scanLine など13種類

## インストール

```bash
pnpm add @suwa-sh/mui-fui-theme
# or
npm install @suwa-sh/mui-fui-theme
```

## 基本的な使い方

### テーマの適用

```tsx
import { ThemeProvider, CssBaseline } from '@mui/material';
import { createFuiTheme } from '@suwa-sh/mui-fui-theme';

function App() {
  const theme = createFuiTheme('dark'); // 'dark' | 'light'

  return (
    <ThemeProvider theme={theme}>
      <CssBaseline />
      <YourApp />
    </ThemeProvider>
  );
}
```

### カラーとエフェクトの取得

```tsx
import { getColors, getGlowEffects, getStageColors } from '@suwa-sh/mui-fui-theme';

// テーマカラー（FuiColors型）
const colors = getColors('dark');
colors.primary           // プライマリカラー（ダーク: #FFB300, ライト: #1976D2）
colors.text.accent       // アクセントカラー
colors.text.primary      // プライマリテキスト
colors.background.paper  // ペーパー背景色
colors.background.elevated // 浮き上がった背景色
colors.background.input  // 入力フィールド背景

// グローエフェクト
const glowEffects = getGlowEffects('dark');
glowEffects.soft    // 控えめな光
glowEffects.medium  // 中程度の光
glowEffects.strong  // 強い光
glowEffects.text    // テキスト用

// ステージカラー（13段階）
// stage1-4: テーマカラーのグラデーション
// stage4→13→1: 色相環（Deep Orange → Red → ... → Green → stage1に戻る）
const stageColors = getStageColors('dark');
stageColors.stage1  // #FFB300 Amber（ダーク）/ #1976D2 Blue（ライト）
stageColors.stage4  // #FF6F00 Deep Orange（色相環の起点）
stageColors.stage13 // #4CAF50 Green
// ...stage13まで対応
```

### コンポーネントの使用

```tsx
import {
  AppGrid,
  DiamondNode,
  IconBox,
  ProgressBar,
  StatusIndicator,
  NavMenuItem,
  SectionHeader,
  ColorLegend,
} from '@suwa-sh/mui-fui-theme';

// グリッドレイアウト
<AppGrid container spacing={2}>
  <AppGrid size={{ xs: 12, md: 6 }}>Content</AppGrid>
</AppGrid>

// ダイヤモンドノード
<DiamondNode color={stageColors.stage1} size={60}>
  <CheckIcon sx={{ color: stageColors.stage1 }} />
</DiamondNode>

// アイコンボックス
<IconBox color={stageColors.stage1} size={48}>
  <SettingsIcon />
</IconBox>

// 進捗バー
<ProgressBar
  value={75}
  color={stageColors.stage3}
/>

// ステータスインジケータ
<StatusIndicator status="success" label="Active" />
```

### アニメーションの使用

```tsx
import { fadeInUp, pulse, scanLine, glowPulse } from '@suwa-sh/mui-fui-theme';

<Box sx={{ animation: `${fadeInUp} 0.5s ease-out` }}>
  Fade in content
</Box>
```

### Hooksの使用

```tsx
import { useTextDecode, useScrollAnimation } from '@suwa-sh/mui-fui-theme';

// Matrix/JARVIS風テキストデコード
function DecodedTitle() {
  const { displayText, isComplete } = useTextDecode('SYSTEM ONLINE', {
    startDelay: 500,  // アニメーション開始までの遅延(ms)
    duration: 2000,   // アニメーション全体の時間(ms)
  });
  return <Typography>{displayText}</Typography>;
}

// スクロールトリガーアニメーション
function AnimatedSection() {
  const { ref, isVisible } = useScrollAnimation({ threshold: 0.1 });
  return (
    <Box ref={ref} sx={{ opacity: isVisible ? 1 : 0 }}>
      Content
    </Box>
  );
}
```

## エクスポート一覧

### テーマ関連

| エクスポート | 説明 |
|------------|------|
| `createFuiTheme` | MUIテーマを生成 |
| `createFuiCustomTheme` | レスポンシブ対応のカスタムテーマを生成 |
| `getColors` | テーマモード別のFuiColorsを取得 |
| `getThemeColors` | getColorsのエイリアス |
| `getStageColors` | ステージカラー（13段階）を取得 |
| `getGlowEffects` | グローエフェクトを取得 |
| `createFuiCodeMirrorTheme` | CodeMirror用テーマを生成 |

### コンポーネント

#### Layout
| コンポーネント | 説明 |
|--------------|------|
| `AppGrid` | MUI Grid2のラッパー（レスポンシブレイアウト） |

#### Atoms
| コンポーネント | 説明 |
|--------------|------|
| `NavMenuItem` | ナビゲーションメニューアイテム |
| `ProgressBar` | アニメーション付き進捗バー |
| `ColorLegend` | カラー凡例 |
| `StatusIndicator` | ステータスインジケータ |
| `SectionHeader` | セクションヘッダー |

#### Molecules
| コンポーネント | 説明 |
|--------------|------|
| `DiamondNode` | ダイヤモンド型ノード |
| `IconBox` | アイコンボックス |

### アニメーション

| アニメーション | 説明 |
|--------------|------|
| `fadeInUp`, `fadeInLeft`, `fadeInRight` | フェードイン |
| `slideInDiagonal` | 斜めスライドイン |
| `scaleIn`, `rotateIn` | スケール・回転 |
| `pulse`, `glowPulse` | パルス |
| `float`, `hologramFlicker` | フロート・フリッカー |
| `scanLine`, `dataStream`, `gridPulse` | HUD/サイバー |

### Hooks

| Hook | 説明 |
|------|------|
| `useTextDecode` | Matrix/JARVIS風テキストスクランブル |
| `useScrollAnimation` | スクロールトリガーアニメーション |

### Logger

| エクスポート | 説明 |
|------------|------|
| `createLogger` | カスタムロガーを生成 |
| `logger` | デフォルトロガーインスタンス |
| `showBanner` | FUI Themeのバナーを表示 |

### 型定義

| 型 | 説明 |
|---|------|
| `ThemeMode` | `'dark' \| 'light'` |
| `FuiColors` | テーマカラー型 |
| `FuiCustomTheme` | カスタムテーマ型 |
| `ResponsiveValue` | レスポンシブ値型 |
| `UseTextDecodeOptions` | useTextDecode オプション型 |
| `UseTextDecodeResult` | useTextDecode 戻り値型 |
| `UseScrollAnimationOptions` | useScrollAnimation オプション型 |
| `UseScrollAnimationResult` | useScrollAnimation 戻り値型 |
