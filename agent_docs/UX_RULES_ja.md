# 松下村塾 UX心理学用語集
Source: https://www.shokasonjuku.com/ux-psychology

## 概要
ユーザーエクスペリエンス（UX）を向上させるために重要な、意思決定に影響を与える心理学効果、バイアス、デザイン原則の定義と概要。

## 原則リスト

* **美的ユーザビリティ効果 (Aesthetic-Usability Effect)**
  美しくデザインされたプロダクトは、そうでないものより使いやすいと直感的に感じられる現象。
* **アンカー効果 (Anchor Effect)**
  最初に提示された情報（アンカー）が基準となり、その後の判断や評価に影響を与える心理現象。
* **バナー・ブラインドネス (Banner Blindness)**
  Webサイト上の広告バナーのような要素を、ユーザーが無意識に無視してしまう現象。
* **認知負荷 (Cognitive Load)**
  ユーザーが情報を処理・理解するために消費する精神的なエネルギー量。
* **確証バイアス (Confirmation Bias)**
  自分の信念や仮説に合う情報を優先的に受け入れ、矛盾する情報を無視・否定する傾向。
* **好奇心ギャップ (Curiosity Gap)**
  情報の欠如（ギャップ）を埋めたいという欲求を利用し、ユーザーの興味や行動を喚起する手法。
* **決断疲れ (Decision Fatigue)**
  繰り返しの決断により精神的エネルギーが消耗し、合理的な選択が困難になる現象。
* **おとり効果 (Decoy Effect)**
  あえて魅力の低い「おとり」の選択肢を混ぜることで、特定の選択肢をより魅力的に見せる誘導手法。
* **デフォルト効果 (Default Bias)**
  明確な変更理由がない限り、提示された初期設定（デフォルト）のまま選択し続ける傾向。
* **ドハティの閾値 / 0.4秒の壁 (Doherty Threshold)**
  応答時間が0.4秒を超えるとユーザーの興味喪失リスクが高まるという閾値。
* **共感ギャップ (Empathy Gap)**
  作り手がユーザーの感情やニーズ、立場に対して十分に共感できない乖離状態。
* **授かり効果 (Endowment Effect)**
  自分が既に所有しているものを、持っていないものよりも高く評価する傾向。
* **期待バイアス (Expectation Bias)**
  事前の期待や先入観が、実際の体験や評価に影響を与える現象。
* **親近性バイアス (Familiarity Bias)**
  過去に見聞きした経験があるデザインや機能に対し、好感や親しみを感じる傾向。
* **段階的要請 / フット・イン・ザ・ドア (Foot in the Door Effect)**
  小さなお願いを承諾させた後だと、より大きなお願いも受け入れられやすくなる現象。
* **フレーミング効果 (Framing)**
  情報の内容そのものより、提示方法（枠組み・表現）によって判断が左右される現象。
* **ゲーミフィケーション (Gamification)**
  ゲームの要素（報酬、競争など）を取り入れ、モチベーション向上や行動を促す手法。
* **目標勾配効果 (Goal Gradient Effect)**
  ゴールが近づくにつれて、達成に向けた行動や努力が加速する心理現象。
* **ハロー効果 (Halo Effect)**
  ある対象の目立つ特徴（良い印象など）に引きずられ、全体の評価が歪められる現象。
* **観察効果 / ホーソン効果 (Hawthorne Effect)**
  人に見られている（観察されている）と自覚することで、行動が変わる傾向。
* **意図的な壁 (Intentional Friction)**
  誤操作防止や熟考を促すために、あえて操作の妨げや遅延を設けるデザイン手法。
* **労働の錯覚 (Labor Illusion)**
  処理に労力や時間がかかっているように見せることで、サービスの価値を高く感じさせる現象。
* **損失回避 (Loss Aversion)**
  利益を得る喜びよりも、損失を避けることを優先する心理傾向。
* **ナッジ効果 (Nudge)**
  強制することなく、ユーザーの自発的な行動変容を促すように設計された「肘でつつくような」介入。
* **ピーク・エンドの法則 (Peak-End Rule)**
  体験の評価は、全体平均ではなく「最も感情が動いた時（ピーク）」と「最後（エンド）」で決まるという法則。
* **プライミング効果 (Priming)**
  先行する刺激（情報）が、その後の無意識的な判断や行動に影響を与える効果。
* **段階的開示 (Progressive Disclosure)**
  一度に全ての情報を出さず、必要に応じて段階的に情報や機能を表示し、認知負荷を下げる手法。
* **ピグマリオン効果 (Pygmalion Effect)**
  他者からの期待を受けることで、その期待通りに成果を出そうとする心理的行動変容。
* **誘導抵抗 (Reactance)**
  選択の自由を奪われたり制限されたと感じた時に生じる心理的な反発。
* **反応型オンボーディング (Reactive Onboarding)**
  ユーザーの実際の行動や状況に応じて、適切なタイミングで提示される操作ガイド。
* **希少性効果 (Scarcity)**
  数や期間が限定されているものに対し、より高い価値を感じて欲求が高まる心理効果。
* **選択的注意 (Selective Attention)**
  多くの情報の中から、自分にとって重要な情報だけに注意を向け、他を無視する機能。
* **系列位置効果 (Serial Position Effect)**
  リストの中間よりも、最初（初頭効果）と最後（親近効果）の項目が記憶に残りやすい現象。
* **スキューモーフィズム (Skeuomorphism)**
  現実世界の物質的な質感や特徴をデジタル上で模倣し、直感的な理解を促すデザイン手法。
* **社会的証明 (Social Proof)**
  自分の判断より、他者の行動や評価（レビューや実績など）を信頼し、同調する心理現象。
* **サンクコスト効果 (Sunk Cost Effect)**
  既に費やしたコスト（金・時間）を取り戻そうとして、非合理的な継続・投資をしてしまう心理。
* **調査バイアス (Survey Bias)**
  調査設計や回答状況により、結果が実態と乖離したり偏りが生じたりすること。
* **誘惑の結びつけ (Temptation Bundling)**
  「やりたいこと（快楽）」と「やるべきこと（苦痛）」をセットにし、行動実行率を高める手法。
* **ユーザー歓喜効果 (User Delight)**
  機能性や利便性を超え、ユーザーに予期せぬ喜びや驚きを与える体験。
* **変動型報酬 (Variable Reward)**
  報酬のタイミングや内容をランダムにすることで、期待感を煽り行動を継続させる仕組み。
* **ビジュアル・アンカー (Visual Anchor)**
  視覚的に目立つ要素を配置し、ユーザーの視線や注意を特定箇所に引きつける手法。
* **視覚的階層 (Visual Hierarchy)**
  大きさ、色、配置などを調整し、情報の重要度や優先順位を視覚的に整理する手法。
* **ツァイガルニク効果 (Zeigarnik Effect)**
  完了したタスクよりも、中断されたり未完了のタスクの方が強く記憶に残る現象。
