# FUI Theme

**FUI (Futuristic User Interface) Theme** は、MUI (Material-UI) をベースとした未来的なデザインシステムです。

## 特徴

- **シャープなエッジ**: 角丸を排除したシャープなデザイン
- **グローエフェクト**: ダークモードで映える光るエフェクト
- **モノスペースフォント**: JetBrains Mono を採用したテクニカルな雰囲気
- **ステージカラー**: ワークフローの段階を表現する色相環ベースのカラースキーム
- **ダーク/ライトモード対応**: 両モードで最適化されたカラーパレット
- **レスポンシブ対応**: ブレークポイント別のサイズ定義

## インストール

```bash
pnpm add @suwa-sh/mui-fui-theme
```

## 基本的な使い方

### テーマの適用

```tsx
import { ThemeProvider } from '@mui/material/styles';
import CssBaseline from '@mui/material/CssBaseline';
import { createFuiTheme } from '@suwa-sh/mui-fui-theme';

function App() {
  const theme = createFuiTheme('dark'); // 'dark' | 'light'

  return (
    <ThemeProvider theme={theme}>
      <CssBaseline />
      <YourApp />
    </ThemeProvider>
  );
}
```

### カラーとエフェクトの取得

```tsx
import { getThemeColors, getGlowEffects, getStageColors } from '@suwa-sh/mui-fui-theme';

// テーマカラー
const themeColors = getThemeColors('dark');
themeColors.text.accent   // アクセントカラー
themeColors.text.primary  // プライマリテキスト
themeColors.background.paper // 背景色

// グローエフェクト
const glowEffects = getGlowEffects('dark');
glowEffects.soft    // 控えめな光
glowEffects.medium  // 中程度の光
glowEffects.strong  // 強い光

// ステージカラー（ワークフロー用）
const stageColors = getStageColors('dark');
stageColors.stage1  // 第1段階
stageColors.stage2  // 第2段階
stageColors.stage3  // 第3段階
stageColors.stage4  // 第4段階
// ...stage12まで対応
```

### コンポーネントの使用

```tsx
import {
  AppGrid,
  DiamondNode,
  IconBox,
  ProgressBar,
  StatusIndicator,
} from '@suwa-sh/mui-fui-theme';

// グリッドレイアウト
<AppGrid>
  <Box>Content</Box>
</AppGrid>

// ダイヤモンドノード
<DiamondNode
  icon={<CheckIcon />}
  label="Step 1"
  status="success"
/>

// アイコンボックス
<IconBox
  icon={<SettingsIcon />}
  color={stageColors.stage1}
/>

// 進捗バー
<ProgressBar
  value={75}
  color={stageColors.stage3}
/>

// ステータスインジケータ
<StatusIndicator status="success" label="Active" />
```

## エクスポート一覧

### テーマ関連

| エクスポート | 説明 |
|------------|------|
| `createFuiTheme` | MUIテーマを生成 |
| `createFuiCustomTheme` | レスポンシブ対応のカスタムテーマを生成 |
| `getThemeColors` | テーマモード別のカラーを取得 |
| `getStageColors` | ステージカラーを取得 |
| `getGlowEffects` | グローエフェクトを取得 |
| `createFuiCodeMirrorTheme` | CodeMirror用テーマを生成 |
