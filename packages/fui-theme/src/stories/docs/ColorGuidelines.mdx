# カラーガイドライン

FUI Theme のカラーシステムについて解説します。

## プライマリカラー

## ステージカラー（13段階）

ワークフローの進行状況を表現するための13段階カラーです。

### カラー構造

- **stage1-4**: テーマカラーのグラデーション
- **stage4→13→1**: 色相環を形成（stage13からstage1に戻って一周）

## グローエフェクト

ダークモードでのみ有効なグローエフェクトです。

| レベル | ダークモード | ライトモード |
|--------|------------|------------|
| Soft | 0 0 10px rgba(255, 179, 0, 0.3) | none |

| Strong | 0 0 30px rgba(255, 179, 0, 0.5) | none |

## 使用方法

### getColors() の使用

```tsx
import { getColors, type ThemeMode } from '@suwa-sh/mui-fui-theme';
import { useTheme } from '@mui/material';

function MyComponent() {
  const theme = useTheme();
  const mode = theme.palette.mode as ThemeMode;
  const colors = getColors(mode);

  return (
    <Box
      sx={{
        color: colors.text.primary,
        backgroundColor: colors.background.paper,
        borderColor: colors.border,
      }}
    >
      Styled content
    </Box>
  );
}
```

### getStageColors() の使用

```tsx
import { getStageColors, type ThemeMode } from '@suwa-sh/mui-fui-theme';

function WorkflowStep({ stage }: { stage: number }) {
  const theme = useTheme();
  const mode = theme.palette.mode as ThemeMode;
  const stageColors = getStageColors(mode);

  // stage1 ~ stage13 まで対応
  const stageKey = `stage${stage}` as keyof typeof stageColors;
  const color = stageColors[stageKey];

  return (
    <Box sx={{ borderColor: color, color }}>
      Step {stage}
    </Box>
  );
}
```

### getGlowEffects() の使用

```tsx
import { getGlowEffects, type ThemeMode } from '@suwa-sh/mui-fui-theme';

function GlowingCard() {
  const theme = useTheme();
  const mode = theme.palette.mode as ThemeMode;
  const glowEffects = getGlowEffects(mode);

  return (
    <Card
      sx={{
        boxShadow: glowEffects.soft,
        '&:hover': {
          boxShadow: glowEffects.medium,
        },
      }}
    >
      Hover me
    </Card>
  );
}
```

### FuiColors型の構造

```tsx
interface FuiColors {
  primary: string;
  secondary: string;
  stages: {
    stage1: string;
    stage2: string;
    // ... stage13まで
  };
  background: {
    default: string;
    paper: string;
    elevated: string;
    input: string;
  };
  text: {
    primary: string;
    secondary: string;
    disabled: string;
    accent: string;
  };
  border: string;
  borderBright: string;
  success: string;
  error: string;
  warning: string;
  info: string;
  glow: {
    soft: string;
    medium: string;
    strong: string;
    text: string;
  };
  gridOpacity: number;
}
```

## カラーのアクセシビリティ

### コントラスト比

FUI Theme のカラーは、WCAG 2.1 Level AA のコントラスト比要件を意識して設計されていますが、a11y（アクセシビリティ）よりもデザインの方向性を優先しています。

> **注意**: 本テーマはa11y基準を厳密に満たしていない場合があります。視覚的なデザインを優先しているため、アクセシビリティが重要なプロジェクトでは追加の調整が必要になる場合があります。

### カラーブラインドへの配慮

ステージカラーは、色覚多様性を考慮して選定されていますが、色だけに依存せず、アイコンやラベルと組み合わせて使用することを推奨します。

```tsx
// 推奨: 色 + アイコン + ラベル
<Chip
  icon={<CheckCircle />}
  label="Completed"
  sx={{ color: stageColors.stage13 }}
/>
```
